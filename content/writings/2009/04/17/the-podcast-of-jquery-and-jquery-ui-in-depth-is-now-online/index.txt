今天是 [OSDC](http://osdc.tw/) 第一天，我報名了下午 [Paul Bakaus](http://paulbakaus.com/) 的 "jQuery and jQuery UI in Depth" 教程。

Paul Bakaus 一開始講 object/prototype 的部份，還真像[敝公司](http://handlino.com/)的 JavaScript 面試考題......

今天聽到了一些之前不曉得，或是誤解的部份，大概紀錄一下：

* !!() 會把 closure 回傳的值轉換成 boolean。
* jQuery 1.3 的 live() 實際上是自己做 event delegation，把 event 綁在 document 上。
* jQuery UI 的 drag & drop 有 snapping、z-index control、還有保持 relative/absolute/fixed positioning 的功能。這個厲害。

我把今天的錄音檔放在 Dropbox，有興趣可以下載：[上半場](http://dl.getdropbox.com/u/132028/jQuery%20and%20jQuery%20UI%20in%20Depth%20-%20Paul%20Bakaus%20Part%201.mp3)、[下半場](http://dl.getdropbox.com/u/132028/jQuery%20and%20jQuery%20UI%20in%20Depth%20-%20Paul%20Bakaus%20Part%202.mp3)。
