--- title: "MoreText.js: a jQuery Plugin to generate Lorem Ipsum Text in Chinese" slogan: more-text-js-a-jquery-plugin-to-generate-lorem-ipsum-text-in-chinese created_at: 2010-04-15 22:50:30.662553 +08:00 blog_post: true filter: - erb - maruku --- 前陣子 [gugod](http://gugod.org/) 寫了一個叫做 [MoreText.js](http://more.handlino.com/) 的 jQuery plugin，它會自動在 HTML prototype 裡面塞入指定句數、沒有意義的文字。因為是指定句數，而不是指定字數，所以在排版的時候很有幫助，很容易看出版面在 worst case 會不會壞掉。 因為產生出來的句子挺惡搞的，所以周遭設計師朋友都很喜歡。而且文字是 JavaScript 動態產生，所以設計師也不需要把垃圾文字寫在原始碼裡面，節省許多麻煩事。 [MoreText.js 專案網站](http://more.handlino.com/)
