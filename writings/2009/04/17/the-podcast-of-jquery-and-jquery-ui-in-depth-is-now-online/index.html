今天是 OSDC 第一天，我報名了下午 Paul Bakaus 的 “jQuery and jQuery UI in Depth” 教程。

Paul Bakaus 一開始講 object/prototype 的部份，還真像敝公司的 JavaScript 面試考題……

今天聽到了一些之前不曉得，或是誤解的部份，大概紀錄一下：

  • !!() 會把 closure 回傳的值轉換成 boolean。
  • jQuery 1.3 的 live() 實際上是自己做 event delegation，把 event 綁在 document 上。
  • jQuery UI 的 drag & drop 有 snapping、z-index control、還有保持 relative/absolute/fixed positioning 的功能。這個厲害。
